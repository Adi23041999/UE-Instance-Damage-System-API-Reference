Register Proxy

Registers the object so the system can interact with it

Inputs

In
Exec

Target
Instance Damage Subsystem Object Reference

Object
Soft Object Reference (by ref)

Class Setup for Damage
Boolean

if false we assume the class isn't setup for damage (like DamagableISMC) and we try detect the proxy begin/end play on tick this is more of a workaround to support default ISM/HISM classes

Outputs

Out
Exec
