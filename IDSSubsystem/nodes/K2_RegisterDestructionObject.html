Register Destruction Object

Registers the object as destruction object.
Registration allows for:
- filtering which components and body instances can cause destruction
- custom proxy detection not requiring overlap event generation both on the object and proxies
- definition of how much force is needed to trigger destructible spawn (ie. prevent the player's car from causing destruction at slightest hits)
- simulated destruction for network simulated proxies
